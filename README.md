# 📝 To-Do List Application

## Project Description
The To-Do List Application is a Python-based desktop application developed using Tkinter. It helps users manage their daily tasks by adding, deleting, completing, and storing tasks easily.

## Features
- Add new tasks
- Delete selected tasks
- Mark tasks as completed
- Clear all tasks
- Save tasks automatically
- Load saved tasks when the application starts
- Display date and time
- Show total number of tasks

## Technologies Used
- Python
- Tkinter (GUI Library)
- File Handling

## How to Run the Project
1. Install Python on your computer.
2. Open `main.py` using Thonny or any Python IDE.
3. Run the program.
4. Start adding your tasks.
